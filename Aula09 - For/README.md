# Estrutura de Repetição FOR

> Quando precisamos repetir tarefas o nossa qualidade começa a reduzir pois é muito exaustivo 

> Com laços de repetição o esforço será reduzido, pois a mesma instrução será executada N vezes sempre da mesma forma.

## Sintaxe 

```` java 

for ( variável de controle; condição de permanência no loop ; incremento da variável de controle) { // inicio do escopo de repetição

   //código a ser repetido

}//fim do escopo de repetição


for(int contador=1; contador<=1000;i++){

   //execute varias vezes aqui

}
